Let

Indica que una variable sólo va a estar definida en un bloque en particular, al terminar el bloque la variable deja de existir, esto es muy útil para evitar errores lógicos cuando alteramos una variable que no deberíamos.

Const

Previene que una variable declarada cambie de valor, convirtiéndose efectivamente en una constante.
Recomendable usar contantes para valores que sabemos que no van a cambiar, así se evitan modificaciones inesperadas.
